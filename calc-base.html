Conduit Suggestion Function

  1. Home
  2. Projects

The suggestEletro function aims to suggest an appropriate conduit based on the quantity of specific cables and their gauge (diameter).

1. Obtaining Values

The function retrieves values from HTML elements where the user selects the cable gauge and enters the quantity of cables.

2. Validity Check

It checks if the gauge is selected and if the quantity is greater than zero.

3. Cable Area Calculation

The area of a cable is calculated using the formula for the area of a circle:

Acable = π × (r2)
Where: r = diameter / 2

4. Conduit Options Definition

The function defines a list of available conduits with their sizes and diameters.

5. Suggestion Initialization

A default message is set to indicate that no conduit has been found.

6. Loop to Find the Suitable Conduit

It iterates over the available conduits and calculates the area for each one using the same formula:

Aconduit = π × (r2)
Where: r = diameter / 2

7. Conduit Capacity Check

The function checks if the total cable area fits within a safe filling percentage of the conduit area:

Atotal = Acable × quantity
Ausable = Aconduit × fill percentage

8. Conduit Suggestion

If a suitable conduit is found, the suggestion is updated. If the conduit is 4 inches, an additional message recommends considering a cable tray.

9. Result Update

The suggested conduit is displayed in the result field.

10. Error Message

If the gauge or quantity is not valid, an error message is displayed.
