Voltage Drop Calculation Code Explanation

This code performs voltage drop calculations in electrical circuits, considering various parameters. Let's analyze the steps and formulas used.

1. Cable List

The list variable contains data about cables, where each sublist has:

  1. Column 0: Type of cable (string).
  2. Column 1: Resistance of the cable (in ohms/km).
  3. Column 2: Reactance of the cable (in ohms/km).

2. Cable Selection

When a user selects a value from a <select> element, the code searches for the corresponding value in the list and fills in the resistance and reactance fields:

cableR.value = varCol1; // Resistance
        cableX.value = varCol2; // Reactance

3. Current Calculation

The calcCirc() function calculates the current based on power and voltage:

  1. Current calculation:

    I = P / V

    Where P is the power of the circuit and V is the nominal voltage.

  2. Calculated current:

    Idim = (I * 0.37) + I

4. Voltage Drop Calculation

To calculate the voltage drop, the code uses the following formulas:

  1. Calculation for single-phase circuits:

    ΔVmono = (2 * Idim * (R + jX) * L) / (100 * n * (V / 10))

  2. Calculation for three-phase circuits:

    ΔVtri = (√3 * Idim * (R + jX) * L) / (100 * n * (V / 10))

5. Voltage Drop Assessment

The code compares the voltage drop result with an allowable value (dropTension):

if (ΔV < dropTension) {
            // Acceptable voltage drop
        } else {
            // Excessive voltage drop
        }

Summary

This code calculates voltage drop in electrical circuits, utilizing resistance and reactance of cables, and considering power, voltage, and circuit length. The result is used to evaluate compliance with electrical installation standards.

These calculations are essential for ensuring the safety and efficiency of electrical installations.
